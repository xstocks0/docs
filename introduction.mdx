xStocks0 is an omnichain tokenized equities protocol — the USDT0 model applied to stocks. It enables a single equity token (e.g. xAAPL, xTSLA) to exist natively across Ethereum, Solana, and TON without wrapped representations or custodial bridges.

## How It Works

xStocks0 uses a **lock/mint model** powered by LayerZero V2 messaging:

- **Ethereum** is the canonical chain. The `xStockOFTAdapter` holds a lock vault — tokens transferred out are locked here, not burned.
- **Remote chains** (Solana, TON) mint tokens when a cross-chain message is received, and burn them when sending back.

This preserves a strict supply invariant: the total locked on Ethereum always equals the total minted across all remote chains.

## Key Properties

| Property | Detail |
|---|---|
| Messaging | LayerZero V2 |
| Payload | 82 bytes, big-endian |
| Shared decimals | 8 |
| Replay protection | `(srcEid, nonce)` mapping / NonceRecord PDA |
| Emergency controls | Pause + `emergencyWithdraw` on Ethereum |
